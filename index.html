What is QtOctave?

QtOctave is a front-end for Octave. Octave is a math application similar to Matlab. Octave is a bit tricky, so QtOctave aims to make it use easier using a simple user interface.

This project was started some time ago but it was abandoned. I'm trying to update its source code. You can browse QtOctave's old website at http://qtoctave.wordpress.com

There is a small tutorial in spanish if needed

Installation

First, download QtOctave 

$ unzip file.zip
$ cd QtOctave
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
